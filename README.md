# Odin Recipes

This project is part of The Odin Project's Foundations curriculum.

## About the Project
The goal of this project is to build a basic recipe website to practice the concepts learned in the HTML and Git lessons.

## Skills Demonstrated
By completing this project, I demonstrate understanding of:
* **Git & GitHub:** Repository setup, commit workflow, and pushing via SSH.
* **HTML Structure:** Writing a correctly structured HTML boilerplate.
* **HTML Elements:** Using headers, paragraphs, lists (ordered/unordered), links, and images.
* **File Management:** Organizing project files and directories.
